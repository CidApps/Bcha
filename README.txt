Bcha es un "Softwafe" todavía no muy bien desarrollado que podrá añadir como contraseña a cualquier archivo con el cóidigo.

start Bcha Lite.vbs o start Bcha Completo.vbs.

Bcha fué creado para que otros programadores puedan cambiar su código ya ajustarla a sus preferencias.

El archivo Bcha Lite.vbs es más simple, tiene menos MsgBox.

El archivo Bcha Completo es más largo, más mensajes.
